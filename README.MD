# Задания из колледжа КЦТ (IT-Hub)- колледж цифровых технологий

## Общая информация

В проектах используются следующие технологии:

- **Язык программирования**: Java
- **Фреймворк**: Spring
- **Система управления базами данных**: PostgreSQL
- **Сборщик проектов**: Maven
